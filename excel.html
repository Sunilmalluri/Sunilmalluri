Beginner: Introduction to Excel Formulas

Key Formulas for Data Summaries

Excel formulas are the foundation of data analysis. Here are three essential formulas to get started:

  • SUM: Adds values in a range. Example: =SUM(A1:A10) calculates the total of cells A1 to A10.
  • AVERAGE: Computes the mean of a range. Example: =AVERAGE(B1:B10) finds the average of cells B1 to B10.
  • IF: Performs conditional logic. Example: =IF(C1>100, "High", "Low") labels values above 100 as "High".

Practice: Create a dataset with sales data (e.g., column A for quantities, column B for prices). Use SUM to calculate total sales and IF to flag high-value transactions.

Intermediate: Pivot Tables for Data Analysis

Creating Dynamic Summaries

Pivot tables allow you to summarize and analyze large datasets without formulas. They’re perfect for exploring trends and patterns.

Steps to Create a Pivot Table:

  1. Select your dataset (e.g., sales data with columns: Date, Product, Region, Sales).
  2. Go to Insert > Pivot Table and choose a location (new or existing worksheet).
  3. Drag fields to areas:
    • Rows: Product (to group by product).
    • Columns: Region (to compare across regions).
    • Values: Sales (set to Sum or Average).
  4. Add filters (e.g., Date) to focus on specific time periods.

Practice: Use a pivot table to summarize total sales by product and region. Add a slicer for interactive filtering.

Advanced: Automating Tasks with VBA

Streamlining Repetitive Tasks

Visual Basic for Applications (VBA) lets you automate repetitive Excel tasks, such as formatting or data processing.

Example: Auto-Format a Table

Sub FormatTable()
    Dim ws As Worksheet
    Set ws = ActiveSheet
    With ws.Range("A1:D100")
        .Interior.Color = RGB(200, 200, 200) ' Light gray background
        .Borders.LineStyle = xlContinuous
        .Rows(1).Font.Bold = True
    End With
    MsgBox "Table formatted!"
End Sub

How to Use:

  1. Press Alt + F11 to open the VBA editor.
  2. Insert a new module and paste the code above.
  3. Run the macro to format a table in cells A1:D100.

Practice: Modify the macro to highlight cells with values above a threshold (e.g., Sales > 1000).
