Key Formulas for Data Summaries

Excel formulas are the foundation of data analysis. Here are three essential formulas:

  • SUM: Adds values. Example: =SUM(A1:A10).
  • AVERAGE: Computes mean. Example: =AVERAGE(B1:B10).
  • IF: Conditional logic. Example: =IF(C1>100, "High", "Low").

Practice: Create a sales dataset and use SUM and IF to analyze it.

Basic Data Formatting

Proper formatting improves readability and analysis.

Steps:

  1. Number Formats: Select cells, go to Home > Number Format (e.g., Currency for sales).
  2. Conditional Formatting: Highlight cells (e.g., Home > Conditional Formatting > Highlight Cells Rules to flag values > 100).
  3. Tables: Select data, go to Insert > Table for auto-filtering.

Pivot Tables for Data Analysis

Pivot tables summarize large datasets dynamically.

Steps:

  1. Select data (e.g., Date, Product, Region, Sales).
  2. Go to Insert > Pivot Table.
  3. Drag fields: Product to Rows, Region to Columns, Sales to Values (Sum).
  4. Add Date to Filters for time-based analysis.

VLOOKUP and HLOOKUP

Lookups merge data from multiple tables.

Example: =VLOOKUP(A2, B$2:C$100, 2, FALSE) finds a value in column B and returns the corresponding value from column C.

Steps:

  1. Ensure lookup table is sorted (for approximate match) or use FALSE for exact match.
  2. Write VLOOKUP for vertical lookups or HLOOKUP for horizontal.

Data Cleaning Techniques

Clean data ensures accurate analysis.

Techniques:

  • Text to Columns: Split data (e.g., Data > Text to Columns for "First Last" names).
  • Remove Duplicates: Data > Remove Duplicates to eliminate redundant rows.
  • Flash Fill: Auto-fill patterns (e.g., extract first names).

Advanced Formulas

Complex formulas enable deeper analysis.

  • INDEX/MATCH: =INDEX(B2:B100, MATCH(A2, A2:A100, 0)) for flexible lookups.
  • SUMIFS: =SUMIFS(C2:C100, A2:A100, "North", B2:B100, ">100") for conditional sums.

Automating Tasks with VBA

VBA automates repetitive tasks.

Example: Format Table

Sub FormatTable()
    Dim ws As Worksheet
    Set ws = ActiveSheet
    With ws.Range("A1:D100")
        .Interior.Color = RGB(200, 200, 200)
        .Borders.LineStyle = xlContinuous
        .Rows(1).Font.Bold = True
    End With
    MsgBox "Table formatted!"
End Sub

Steps:

  1. Press Alt + F11, insert a module.
  2. Paste and run the macro.

Advanced VBA for Data Processing

Advanced VBA processes large datasets.

Example: Generate Summary Report

Sub GenerateReport()
    Dim ws As Worksheet, summary As Worksheet
    Set ws = ActiveSheet
    Set summary = Worksheets.Add
    summary.Name = "Summary"
    summary.Range("A1").Value = "Total Sales"
    summary.Range("A2").Value = Application.Sum(ws.Range("C:C"))
    MsgBox "Report generated!"
End Sub

Steps:

  1. Add to VBA module and run.
  2. Check new "Summary" sheet.
