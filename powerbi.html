Connect Power BI to various data sources like Excel, CSV, or databases.

Steps:

  1. Click Home > Get Data > Excel.
  2. Select a file and use the Navigator to choose tables.
  3. Load or transform data in Power Query Editor.

Introduction to DAX Formulas

Data Analysis Expressions (DAX) enable custom calculations.

Examples:

  • SUM: Total Sales = SUM(Sales[Amount])
  • CALCULATE: Sales North = CALCULATE(SUM(Sales[Amount]), Sales[Region] = "North")
  • COUNT: Order Count = COUNT(Sales[OrderID])

Practice: Create a measure for total sales in a specific region.

Advanced

Advanced DAX

Complex DAX measures enable sophisticated analytics.

Example:

YOY Growth = 
VAR CurrentYear = SUM(Sales[Amount])
VAR PreviousYear = CALCULATE(SUM(Sales[Amount]), PREVIOUSYEAR(Sales[Date]))
RETURN
DIVIDE(CurrentYear - PreviousYear, PreviousYear, 0)

Steps:

  1. Create a new measure in the Fields pane.
  2. Use variables for clarity and performance.
  3. Test in a visual (e.g., KPI card).

Practice: Calculate year-over-year sales growth.
