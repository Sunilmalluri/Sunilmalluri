Intermediate

Calculated Fields

Create custom calculations to analyze your data in new ways.

Examples:

  • Profit Ratio: [Profit]/[Sales]
  • Discount Flag: IIF([Discount] > 0, "Discounted", "Full Price")
  • Sales Growth: (SUM([Sales]) - LOOKUP(SUM([Sales]), -1)) / ABS(LOOKUP(SUM([Sales]), -1))

Advanced

Level of Detail (LOD) Expressions

LOD expressions give precise control over the granularity of calculations.

Examples:

// Fixed LOD: Sales per customer regardless of view dimensions
{FIXED [Customer ID] : SUM([Sales])}

// Include LOD: Sales per region within each category
{INCLUDE [Region] : SUM([Sales])}

// Exclude LOD: Sales per category ignoring region
{EXCLUDE [Region] : SUM([Sales])}

Table Calculations

Table calculations perform computations based on the table structure.

Examples:

Steps:

  1. Add a calculation to the view (e.g., SUM(Sales)).
  2. Right-click the measure and select Add Table Calculation.
  3. Choose calculation type and addressing.
